Project Details

Max – Online Cinema Platform

Role: Full Stack Developer
Stack: Django, React, PostgreSQL, TMDb API, Docker
Year: 2025

About Project

Max is a complete streaming platform where users can watch movies and TV shows online. It features smart recommendations, personal watchlists, ratings, trailers, search, and user accounts. The project integrates with external movie APIs and offers a beautiful, responsive UI for all devices.

Key Features

  • Browse and watch thousands of movies and shows
  • Integration with TMDb API for dynamic content updates
  • Personalized watchlists and recommendations
  • Search by title, genre, actor, or release year
  • User authentication and profile settings
  • Rating and comment system
  • Trailer previews and detailed film pages
  • Admin panel for content moderation and management
  • REST API for mobile applications
  • Responsive design with dark/light mode toggle
